Analysis Summary

Summary

Functional analysis can be a powerful tool for summarizing differential expression results from a variety of experiments, including both bulk and single-cell RNA-seq.

What weâ€™ve accomplished:

  • Discussed the concepts behind ORA, GSEA, and Pathway Topology approaches to functional enrichment approaches.
  • Used a web-browser tool to perform ORA analysis of bulk RNA-seq data.
  • Used R/RStudio and WebGestaltR to run ORA for bulk RNA-seq data in a more reproducible manner.
  • Used WebGestaltR to run GSEA for single-cell RNA-seq data
  • Created customized visualizations at the gene and term level
  • Reviewed biological knowledge bases commonly used in functional analyses.
